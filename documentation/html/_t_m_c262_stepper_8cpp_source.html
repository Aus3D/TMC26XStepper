00536 /*
00537  * In a constant off time chopper scheme both coil choppers run freely, i.e. are not synchronized. 
00538  * The frequency of each chopper mainly depends on the coil current and the position dependant motor coil inductivity, thus it depends on the microstep position. 
00539  * With some motors a slightly audible beat can occur between the chopper frequencies, especially when they are near to each other. This typically occurs at a 
00540  * few microstep positions within each quarter wave. This effect normally is not audible when compared to mechanical noise generated by ball bearings, etc. 
00541  * Further factors which can cause a similar effect are a poor layout of sense resistor GND connection.
00542  * Hint: A common factor, which can cause motor noise, is a bad PCB layout causing coupling of both sense resistor voltages 
00543  * (please refer to sense resistor layout hint in chapter 8.1).
00544  * In order to minimize the effect of a beat between both chopper frequencies, an internal random generator is provided. 
00545  * It modulates the slow decay time setting when switched on by the RNDTF bit. The RNDTF feature further spreads the chopper spectrum, 
00546  * reducing electromagnetic emission on single frequencies.
00547  */
00548 void TMC262Stepper::setRandomOffTime(char value) {
00549         if (value) {
00550                 chopper_config_register |= RANDOM_TOFF_TIME;
00551         } else {

00553         }
00554         //if started we directly send it to the motor
00555         if (started) {
00556                 send262(driver_control_register_value);
